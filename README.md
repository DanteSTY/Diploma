Точный расчет структурной надежности сети с ненадёжными рёбрами является классической NP–трудной задачей. 
При увеличении размерности сети количество возможных состояний системы растет экспоненциально, что делает прямое перечисление реализаций случайного графа вычислительно невозможным.


В рамках стандарта OpenMP были разработаны две конкурирующие парадигмы параллельных вычислений:
1) Классическая иерархическая модель, использующая синхронизацию taskwait и реализующая сбор результатов снизу вверх на обратном ходе рекурсии.
2) Асинхронная модель без обратного хода рекурсии, построенная на принципе «сверху вниз». Модель применяет прямую передачу накопленной вероятности и концепцию fire-and-forget,
полностью исключая явные барьеры синхронизации между родительскими и дочерними задачами.


Низкоуровневые оптимизации:

Устранение эффекта ложного разделения кэш-линий (False Sharing) с помощью стратегии структурного выравнивания данных (Padding). 

Использование компактного хранения графов CSR или КАО-ФО(Компактное Адресное Описание — Формат Отношений).

Методы топологической редукции графа:

Удаление висячих вершин (Shoots Deletion)
Последовательно-параллельные преобразования (PST)
Отсечение по базовым графам (Base Graphs Cutoff)

Эвристические подходы к выбору ребра ветвления:

1)Комбинированный поиск с приоритетом треугольников-Особенность данной эвристики заключается в целенаправленном использовании циклов длины 3 (треугольников) для искусственной генерации параллельных ребер, 
что в дальнейшем позволяет применять последовательнопараллельные преобразования (PST) за 𝑂(1) операций.

2)Жадный алгоритм быстрого выбора(Как только встречается любое ребро с суммой степеней < 7 , алгоритм берет его «не глядя», полностью игнорируя ресурсоемкую проверку на наличие циклов длины 3)



Реализован и протестирован подход с использованием побитовых масок (Bitmasking), который за счет 100% кэш-локальности и применения аппаратных инструкций
процессора (BitScan) позволил ускорить базовые операции над графами до 3 раз по сравнению с представлением в виде КАО–ФО массивов.
